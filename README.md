# Агрегатор СМИ2
Агрегатор СМИ2 - точка сбора самых интересных и актуальных новостей российских онлайн-медиа. Картина дня формируется автоматически на базе популярности материалов у читателей. В проекте участвуют русскоязычные издания.

# Страницы
### Главная и новостные топики
Отображаются новости с ранжированием по популярности у читателей. При клике на новость в новой вкладке открывается источник. Топики выбираются в левом вертикальном меню сайта. С интервалом 1 мин проверяются новые статьи на сайте и отображается соответствующее уведомление (зеленая кнопка поверх статей). При нажатии на уведомление новости перезагружаются, а новые новости помечяются знаком "новое".
```
/
/topic/xxx
```
![](https://monosnap.com/file/7e2vesjWBIodH5IhzNK9uCU25m7b0Y.png)

### Региональные новости
Отображаются новости выбранного региона. В случае первого посещения сайта - гео определяется автоматически на основании IP адреса клиента.
```
/region/xxx
```

### Выбор источников
На данной странице у пользователя есть возможность убрать некоторые источники из отображения в ленте новостей.
```
/sources
```
![](https://monosnap.com/file/oc7A1JOZ8YqrD8bXai863QFUnSbfIQ.png)

### Новости 1 источника
При нажатии на иконку источника пользователь попадает на страницу, сотавленную из новостей только этого ресурса.
```
/source/xxx
```

### Выбор региона
Если пользователь кликает на название региона в левом меню, он попадает в список регионов где он может выбрать верный (если неверно определилось автоматически).
```
/regions
```
![Alt text](https://monosnap.com/file/6uA9ofduPGp6Q6H0qgwwFKx8JVyH1s.png)

### Дополнительные возможности
При нажатии на кнопку "Шрифт" меняется размер шрифта 0 -> +1 -> +2 -> +3 -> -1 -> 0.
Пункт меню "О проекте" вызывает показ popup окна с информацией о проекте.
